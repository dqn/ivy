# Visual Novel Engine Comparison

A comparison of ivy with major visual novel engines.

## Engines Overview

| Engine | Language | Platforms | Notes |
|--------|----------|-----------|-------|
| Ren'Py | Python | Win/Mac/Linux/Android/iOS/Web | Most popular, 23,000+ titles |
| TyranoBuilder | JS | Win/Mac/Android/iOS/Web | Drag & drop, Live2D support |
| Naninovel | C# | Unity platforms | Unity integration, highly extensible |
| Visual Novel Maker | JS | Win/Mac/Linux/Android/iOS/Web | Live2D lip sync |
| KiriKiri | TJS | Windows | Popular in Japan, 6,800 titles |
| Monogatari | JS | Web | HTML5 focused, offline support |
